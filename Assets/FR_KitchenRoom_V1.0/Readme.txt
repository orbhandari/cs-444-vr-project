- 支持桌面（PC、Mac）
- 支持移动端（Android、iOS）

- .NET 2.0及.NET 2.0以上

- 后期处理堆栈设置

FR_Kitchenroom_V1.0 presents you with a Nordic kitchen house, complete in every detail.

The scene is highly optimized and is well suited to a variety of platform experiences.

All the furniture and props are exquisitely detailed and can be reused in other projects.





Function:

- 90+ game prefabs

- 2K textures

- Most Of Model Vertice count ranging between 200 and 5000 vertices

- Map types include diffuse, metal roughness, normals, ambient occlusion, and four types of map

- Colliders included

- Desktop support (PC, Mac)

- Support mobile terminal (Android, iOS)

- DX11 and above

- .NET 2.0 and above

- Standard 3D version

- Supports baking and real-time GI

- Post-processing stack Settings


Other:

1. The project color space is linear space. Please adjust the color space to linear space after opening the project

2. If the screen is different from the store resource, check whether the baking information is missing. Please re-bake the lighting

3. Check if you forgot to install Post Processing stack, please install it in package Manager
